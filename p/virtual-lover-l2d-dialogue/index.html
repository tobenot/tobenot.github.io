“请你为我摘星星”
“你要星星吗？我虽然摘不到星星，但如果我画一个给你，那算不算摘到半个星星呢？”
“你今天很漂亮”
“哎呀，这么夸我，害羞了都！谢谢你的夸奖，这身打扮特别为今天准备的呢~”
以上为虚拟恋人课设答辩时实机演示的对话

基于GPT接口做的一个课设，欢迎交流，源码为小组共有，暂不开源~😊
我做了L2D（模型与动作表情接入）💡、长记忆🧠、思维链🔗、人设Prompt🤖、好感度调控的部分💖。
使用Java语言做系统框架，C++生成DLL，通过JNI调用展示L2D形象。
用IDEA、Visual Studio开发，l2d接入cubism sdk。

在大模型能力上套个壳，增强大模型能力。

工程问题

打开工程后无法运行，错误: 找不到或无法加载主类 Main

原因: java.lang.ClassNotFoundException: Main
解决方法：右键src文件夹，Mark Directory as Sources Root
