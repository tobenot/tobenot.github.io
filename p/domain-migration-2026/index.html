域名迁移与项目访问地址变更通知

Domain Transition Notice — 各项目访问地址已迁移至专属子域名，请更新书签与引流链接。

为什么要改

tobenot.top 主站长期以「子目录」形式托管多个独立项目（如 /Epitaph/、/Bedtime-Stories-with-AI-2/）。随着项目增多，子目录名变长、职责边界模糊，也不利于单独维护与分享。

本次调整将各项目迁至专属子域名，并统一门户结构。主站 tobenot.top 继续承担博客与内容发布；作品集、工具、游戏等各有独立入口。

墓志铭路由变更（重要）

墓志铭除域名变更外，路由方式也已调整：

旧版新版
域名tobenot.top/Epitaph/e.tobenot.top
路由Hash 路由：#/路径History 路由：/路径
示例https://tobenot.top/Epitaph/#/novel/xxxhttps://e.tobenot.top/novel/xxx

若你收藏或外链了墓志铭的 Hash 链接，请将 #/路径 改为 /路径，并换用 e.tobenot.top 域名。

VRC 教程与 Cloudflare Pages

原托管在 Cloudflare Pages 的 VRChat 教程（如 vrchat-world-tutorial.pages.dev）默认域名保持不变。

建议通过以后会开发的 vrc.tobenot.top 进入，由门户页统一跳转至各教程与作品。

注意事项

  1. 旧链接重定向：主站 tobenot.top 已为下表全部旧子目录配置跳转（根路径直达 + 深层路径经 404 页转发）。请优先使用新域名书签。
  2. 墓志铭 Hash 链接：访问 tobenot.top/Epitaph/#/路径 会自动跳到 e.tobenot.top/路径；站外手写的外链请直接改用新域名。
  3. 主站 tobenot.top：博客、文章、RSS 等仍在主站，不受影响。
  4. AI 睡前故事本地存档：由于浏览器按域名隔离存储，新站 ai.tobenot.top 首次打开时会自动尝试跨域获取旧站数据并无缝合并。如果自动迁移因浏览器限制未生效，可使用 旧站存档迁移页 手动导出 JSON 后再到新站导入。
