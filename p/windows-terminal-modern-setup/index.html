二、把默认终端设为 PowerShell 7

配好之后建议把默认配置文件切过去，省得每次手动切换。

打开 Windows Terminal（开始菜单搜 “终端”，或 Win + R 输入 wt 回车），点顶部标签栏旁的向下箭头（∨）→ 选 “设置”（或按 Ctrl + ,），左侧点 “启动”，在右侧 “默认配置文件” 下拉里选带深蓝色图标的 “PowerShell” ，保存关闭。

避坑点：要选图标是深蓝色的那个，那才是 PS7。

其他两种快速启动方式：

  • Win + R 运行窗口：输入 pwsh 回车。
  • 文件夹里直接开：在文件管理器地址栏输入 pwsh 回车，会在当前路径下启动 PS7。

三、实用功能速查
