一键打开工作需要的程序和网页
  1. 批处理打开程序和网页
  2. 开机启动

本文将介绍如何创建一个简单而实用的批处理文件，通过一键启动，快速打开工作所需的程序和网页。

批处理打开程序和网页

新建一个txt，改后缀bat
找到浏览器、应用的本地地址，这个在快捷方式右键属性看“目标”就行了

@echo off
chcp 65001 > nul

echo 正在启动工作环境...

rem 启动Chrome浏览器并打开收藏夹和歌单
start "" "C:\Program Files (x86)\Google\Chrome\Application\chrome.exe" "https://space.bilibili.com/你的歌单" "https://www.bilibili.com/video/BV1PC4y1J75R/"

rem 启动OneNote
start "" "C:\Program Files\Microsoft Office\root\Office16\ONENOTE.EXE"

rem 启动QQ
start "" "C:\Program Files (x86)\Tencent\QQ\Bin\QQScLauncher.exe"

echo 工作环境已准备就绪！
pause >nul
exit

这时候运行这个批处理就能打开所有东西了。

这一行能让它等1秒

timeout /nobreak /t 1 >nul

开机启动

适用于Windows系统：

  1. 找到启动文件夹：

    • 使用快捷键 Win + R 打开运行对话框。
    • 输入 shell:startup 并按 Enter 键。

  2. 将批处理文件复制到启动文件夹：

    • 将你的批处理文件复制到打开的启动文件夹中。

  3. 设置启动文件夹权限：

    • 确保你对启动文件夹有写入权限，以便在那里创建或复制文件。

  4. 验证设置：

    • 重新启动计算机，验证批处理文件是否在系统启动时运行。
